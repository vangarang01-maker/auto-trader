# 전략 V1 — 피터 린치 성장주 스크리닝

## 개요

피터 린치의 고성장-저평가 원칙을 적용한 퀀트 전략.
순이익·매출이 빠르게 성장하면서도 성장성 대비 저평가(PEG ≤ 1)된 KOSPI 종목을 선별한다.

---

## 스크리닝 파이프라인

### 1단계 — DART 재무 스크리닝

소스: `src/screening/fundamental.py` — `FundamentalScreener.screen_all()`

| 지표 | 조건 | 비고 |
|------|------|------|
| 당기순이익 | > 0 | 흑자 기업 |
| 순이익 성장률 | ≥ 20% | YoY |
| 매출 성장률 | ≥ 10% | YoY |
| 부채비율 | ≤ 50% | 보수적 재무 건전성 |
| 영업현금흐름 | > 0 | 데이터 없으면 통과 |

- 사업보고서 기준연도: 4월 이후 → 전년도 / 4월 이전 → 전전년도
- 연결재무제표 우선, 없으면 별도재무제표
- KOSPI 전 종목 병렬 조회 (workers=32)

### 2단계 — KIS PEG 필터

소스: `FundamentalScreener.apply_peg_filter()`

```
PEG = PER(KIS 실시간) / 순이익성장률(%)
```

- PEG ≤ 1.0 통과 (성장성 대비 저평가)
- KIS 조회 실패 시 통과 처리

### 3단계 — KOSPI 초과성과 필터

소스: `FundamentalScreener.apply_kospi_outperformance_filter()`

최근 3개월(약 60 거래일) 기준:

```
상승포착률 = mean(종목 등락률 | KOSPI 상승일) / mean(KOSPI 등락률 | KOSPI 상승일) × 100
하락포착률 = mean(종목 등락률 | KOSPI 하락일) / mean(KOSPI 등락률 | KOSPI 하락일) × 100
```

통과 조건: **상승포착률 > 하락포착률**
(예: 상승포착 130%, 하락포착 80% → KOSPI 오를 때 1.3배, 내릴 때 0.8배)

- 공통 거래일 20일 미만 또는 상승·하락일 각 5일 미만 → 제외
- FDR 조회 실패 → 제외 (통과 처리 없음)

### 건강검진 — 7개 지표

소스: `src/screening/health_check.py`

3단계 통과 종목(~10개)에 대해 절대 기준으로 0~100점 산출. DB(`company_health`) 7일 캐시.

| 지표 | 만점 | 최적 기준 | 최악 기준 |
|------|:----:|---------|---------|
| PER | 10 | ≤ 10배 | 50배 |
| PBR | 20 | ≤ 1.0배 | 5.0배 |
| ROE | 10 | ≥ 30% | 0% |
| ROIC | 10 | ≥ 20% | 0% |
| 영업이익률 | 10 | ≥ 30% | 0% |
| 부채비율 | 20 | ≤ 30% | 200% |
| 배당수익률 | 20 | ≥ 4% | 0% |

건강검진 점수 내림차순 상위 5개 선정 → `picks.json`

## 매매 전략 (공통)

| 신호 | 조건 |
|------|------|
| 매수 | RSI-14 < 35 (과매도) |
| 매도 | RSI-14 ≥ 75 (과매수) |
| 익절 | 매수 평단가 대비 +15% |
| 손절 | 매수 평단가 대비 -7% |

최대 보유 5종목, 종목당 200만 원 균등 배분 (총 1,000만 원 기준).

---

## 특징 및 한계

**강점**
- 피터 린치가 검증한 고성장·저평가 원칙의 단순·명확한 기준
- PEG 단일 지표로 성장성 대비 밸류에이션을 포착
- KOSPI 초과성과 필터로 모멘텀 확인

**한계**
- 부채비율 ≤ 50% 조건으로 금융주·지주사 대부분 제외
- 고성장 성장주 특성상 고배당주·가치주는 선별 불가
- 2008년 이후 글로벌 가치 프리미엄 약화 환경에서 성과 제한 가능성
