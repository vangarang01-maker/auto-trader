# 수익률 극대화 개선 이니셔티브

우선순위 순서로 구현. 각 항목 완료 시 ✅ 표시.

---

## 1순위 — 시장 국면 감지 (Market Regime) ✅
> 약세장에서 기계적 매수를 막아 손실 방어

- KOSPI 200일 이동평균 기준 판별
  - 200MA 위 → 강세장: 정상 매매
  - 200MA 아래 → 약세장: 신규 매수 차단 or MAX_HOLD 축소
- 구현 위치: `run_trade.py` 매수 직전 체크
- 부가: 텔레그램 알림에 시장 국면 표시

## 2순위 — 외국인/기관 수급 반영 ✅
> 수급이 받쳐주는 종목이 신호 신뢰도 높음

- KIS API 외국인 순매수 연속일 조회
- 외국인 순매수 지속 종목에 건강검진 보너스 or 우선 매수
- 외국인 대량 순매도 종목 매수 차단

## 3순위 — 진입/청산 신호 고도화 ✅
> RSI 단독 신호의 오류율 감소

- RSI + 거래량 급증 조합 진입 조건 추가
- 익절/손절 고정값(+15%/-7%) → ATR 기반 동적 설정
- 횡보 제한: 매수 후 N일 내 목표가 미달 시 청산

## 4순위 — 실적 발표 캘린더 연동
> 실적 불확실성 구간 리스크 관리

- DART 공시에서 실적 발표 예정일 파악
- 발표 2~3일 전 신규 진입 차단
- 어닝 서프라이즈(실제 > 예상 20%↑) 시 건강검진 보너스

## 5순위 — 포지션 사이징 고도화
> 확신도 높은 종목에 더 많이 배분

- 현재: 5종목 균등 배분(각 200만원)
- 변경: 건강검진 점수 비례 배분
  - 예) 60점/50점/40점 → 3:2.5:2 비율 배분

## 6순위 — 백테스팅
> 전략 파라미터 최적화 및 검증

- 과거 3년치 데이터로 시뮬레이션
- RSI 기준값, PEG 상한, 익절/손절 비율 최적화
- Walk-forward 테스트로 과적합 방지
